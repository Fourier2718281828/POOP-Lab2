Step-by-step explanation

  1. Menubar can be used either to select the tools, that have icons beneath or to make new file or open other files

    2. Press "New" to create new file;

    Press "Open" to open external file;

    Press "Save" to save current file;

    Press "Save as" to save current file with adress clarification;

    Press "Exit" to stop the program;

    All other tools in the following items : View, Tools, Help, are present in the toolbar beneath

  2. Use drawing tool to draw pictures by your hand

    3. Select the tool and draw!

  3. Colour chooser can be used to change colour of the drawing pen.

    4. Click the "Colour chooser" button and choose the colour.

  4. Clearing tool can be used to clear the canvas up

    5. Click the "Clearing tool" button and the canvas will get cleared.

  5. Shape drawing tools

    6. Choose the shape you want

    Then click and move your mouse to draw the shape

    Once you release the mouse button, the shape will be drawn.

  6. Save tools

    7. When you have drawn all you want or just want to save you progress, click "Save" button

    If you haven't saved yet, you will need to choose the location of tour file save.

    If you have saved your progress before, the progress will be automatically saved to pre-chosen location

    In case you want to change the location, click on "Save as" button and choose the path you want.

  7. Scaling tools

    8. If you want to scale your image chose one of the scaling tools

    "+" means zooming-in

    "-" means zooming-out

  8. Help tool

    9. Press "Help" tool button to read the instruction to the program

  9. Exit

    10. Press "Exit" button to shut the program down
